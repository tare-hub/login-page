# login-page
🔐 Login & Registration Page

A simple and responsive Login & Registration system built using HTML, CSS, and JavaScript. This project focuses on clean UI design and basic client-side form validation.

✨ Features
User-friendly login and signup interface
Form validation for:
✅ Username
✅ Email
✅ Password
Responsive design for different screen sizes
Instant validation feedback using JavaScript
🛠 Technologies Used
HTML – Structure
CSS – Styling & layout
JavaScript – Validation & interactivity
📌 Validation Rules
Username must not be empty
Email must follow a valid format
Password must meet minimum length/security requirements
🚀 How to Use
Clone or download the project
Open index.html in your browser
Test the login and registration forms
